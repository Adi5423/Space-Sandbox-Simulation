- Windows 10 or later

- MinGW-w64 compiler
- CMake 3.10 or later

## Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/Space-Sandbox-Simulation.git
   cd Space-Sandbox-Simulation
   ```

2. Run the setup script to download dependencies:
   ```bash
   setup.bat
   ```

3. Build the project:
   ```bash
   build.bat
   ```

4. Run the application:
   ```bash
   build/astro-sim.exe
   ```
